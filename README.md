# microutils

Microutils — это простой Python-пакет с базовыми утилитами: арифметические операции и вспомогательные функции.

## Возможности

* Сложение чисел (add)
* Простые вспомогательные функции (foo)
* Минимальный пример структуры Python-пакета

## Установка

Установка из TestPyPI:
```bash
pip install --index-url https://test.pypi.org/simple/microutils
```
## Использование

Пример:
```python
from microutils import add, foo

print(add(2, 3))   # 5
print(foo(4, 5))   # 9
```
## Структура проекта

* calc.py — функции для вычислений
* example.py — вспомогательные функции
* app.py — пример запуска
